What is jQuery Toolkit ?

jQuery Toolkits provides set of Modules for Web Developers and Web Designers based by jQuery Library
Modules could be Widgets ,Plug-ins, or set of JavaScript utilities.

What does jQuery Toolkit Include

  • You don't have to download the jQuery toolkit to use jQuery Toolkit Modules
  • Each Module might include 0 or more sub Modules.
  • Every Sub Module is a stand alone Module
  • jQuery toolkit Concentrates on real life examples in web developments and tries to break down super complicated apps small reusable app
  • Tutorials on how to write a good jQuery code
  • Toturials on how to write a Object Oriented Javascript code
